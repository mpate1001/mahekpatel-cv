# mahekpatel-cv

Personal portfolio website showcasing my experience, projects, and skills as a Software Engineer and Data Scientist.

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- D3.js
- Framer Motion

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deployment

Deployed on Vercel. Pushes to `main` trigger automatic deployments.
